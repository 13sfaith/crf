## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

To build your own verison of this script you will need.

* Cargo
* Rust
```sh
# if you use rustup update rust using:
rustup update
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/13sfaith/crf.git
```
2. Build the release
```sh
cargo build --release
```
